#To start the project:
    1. Go to https://nodejs.org/ and click 'Install' to install Node.js (Don't download Node.exe)
    2. Open command line in the project root directory
    3. Run `npm install gulp -g`
    3. Run `npm install`
    4. Run `gulp bower`
    5. Run `gulp`
